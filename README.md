# Penguin

Project Name: Penguin
This project is a fun and interactive web page that displays an animated penguin in the foreground and a beautiful scenic background.

Technologies Used:
HTML
CSS
Getting Started:
To view the webpage, simply download the HTML and CSS files and open the HTML file in your preferred web browser.

Functionality:
The webpage displays an animated penguin in the foreground, consisting of a penguin head, face, chin, eyes, blush, and beak, along with a shirt, body, arms, and feet. The penguin is surrounded by a scenic background consisting of a sun, mountains, and sky. Additionally, the ground contains several text messages that appear when the user interacts with the penguin.

Usage:
The webpage is intended for entertainment purposes only and can be used to relax and enjoy the animated penguin and background scenery.
